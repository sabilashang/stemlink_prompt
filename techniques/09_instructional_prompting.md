# Instructional Prompting

## Definition
Provide very detailed step-by-step instructions to control behavior.

## Prompt Structure
```
Follow these steps:
1. Do X
2. Then do Y
3. Finally, output Z
```

## Example (Python)
```python
Follow these steps:
1. Define a list of numbers from 1 to 10
2. Square each number
3. Print the result
```

## When to Use
Use when precision and procedure control matter.
